# DJConnect Windows 3.2.5

- Vervollständigt die app-weite Lokalisierung für Englisch, Niederländisch, Deutsch, Französisch und Spanisch, einschließlich Einstellungen, Meldungen, Onboarding, Mini-Spielen und Leerzuständen.
- Verbindet die Windows-Protokollaktivierung `djconnect://` mit dem vorhandenen QR-/Deeplink-Pairing-Flow.
- Fügt strengere Lokalisierungsprüfungen für fehlende Schlüssel, doppelte Schlüssel, Platzhalterabweichungen und Ressourcenformatierung hinzu.
- Verbessert normale Windows-UI-Texte, damit technische Diagnosen in About, Feedback und Logs bleiben.
