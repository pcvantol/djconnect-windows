# DJConnect Windows 3.2.9

- Nutzt die kanonischen Home Assistant-Routen `/api/djconnect/v1/...` fuer Pairing, Status, Ask DJ, Track Insight, Music DNA und Music Discovery.
- Aktualisiert die QR-/Deeplink-Pairingvalidierung fuer den v1 Pair-Endpunkt, waehrend der Windows-Identity-Vertrag unveraendert bleibt.
- Fuegt eine Regressionspruefung hinzu, damit alte DJConnect API-Routen ohne v1 nicht in Clientcode oder Dokumentation zurueckkehren.
