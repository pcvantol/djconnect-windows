# DJConnect Windows 3.2.3

- Sprachunterstützung für Englisch, Niederländisch, Deutsch, Französisch und Spanisch hinzugefügt.
- Rohe Pairing- und API-Fehlercodes durch kurze, hilfreiche Hinweise ersetzt, inklusive veralteter Authentifizierung, ungültiger Kopplungscodes und Clienttyp-Konflikten.
- Lokalisierungsvalidierung hinzugefügt, damit jede unterstützte Sprache alle String-Schlüssel enthält.
