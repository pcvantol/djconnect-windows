# DJConnect Windows 3.2.5

- Completa la localización de toda la app en inglés, neerlandés, alemán, francés y español, incluidos ajustes, avisos, onboarding, minijuegos y estados vacíos.
- Conecta la activación del protocolo Windows `djconnect://` con el flujo existente de emparejamiento QR/deeplink.
- Añade validación de localización más estricta para claves ausentes, duplicadas, diferencias de marcadores y formato de recursos.
- Pule los textos normales de la interfaz Windows para que los diagnósticos técnicos queden en About, feedback y logs.
