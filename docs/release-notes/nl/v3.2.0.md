# DJConnect Windows 3.2.0

- De Windows client is bijgewerkt naar het DJConnect `3.2.x` protocol.
- Pairing gebeurt nu met de lokale Home Assistant URL en de code die de
  DJConnect integration toont. Windows host geen lokale Client API meer en
  adverteert geen mDNS voor pairing.
- Na lokale pairing kan de app terugvallen van lokale Home Assistant naar een
  ondersteunde remote Home Assistant URL.
- Diagnostiek toont nu HA-verbindingsmodus, remote support, muziekbackend,
  backend revision, target player en backend-capabilities.
- Playback-acties worden backend-owned doorgestuurd, zodat Spotify Direct en
  Music Assistant responses werken zonder Spotify-only aannames.
