# DJConnect Windows 3.2.5

- Maakt app-brede lokalisatie compleet voor Engels, Nederlands, Duits, Frans en Spaans, inclusief instellingen, meldingen, onboarding, mini-games en lege staten.
- Koppelt Windows `djconnect://` protocolactivatie aan de bestaande QR/deeplink pairing-flow.
- Voegt strengere lokalisatievalidatie toe voor ontbrekende keys, dubbele keys, placeholderverschillen en resource-formatting.
- Scherpt gewone Windows UI-teksten aan zodat technische diagnostiek in About, feedback en logs blijft.
