# DJConnect Windows 3.1.2

- Onboarding-gated lokale pairing toegevoegd met Client adres, koppelcode kopieeracties, striktere mDNS-advertising en pairing reset.
- Volledige desktop-schermen toegevoegd voor Speelt Nu, Ask DJ, Wachtrij, Afspeellijsten, Instellingen, Privacy, Logs, Feedback, Crashrapporten, About, Legal, What's New en Mini-games.
- Privacy-safe diagnostics-redaction toegevoegd voor logs, feedback, crashrapporten, clipboard exports en GitHub issue-prefill.
- Session-only Demo Mode toegevoegd met lokale voorbeelddata voor playback, wachtrij, afspeellijsten en Ask DJ.
- Update Required-afhandeling toegevoegd voor protocolmismatches terwijl instellingen, logs, privacy, legal en feedback bereikbaar blijven.
- Wakeword prompt/settings-state toegevoegd achter een feature gate tot er een echte foreground listener beschikbaar is.
- Non-functional requirement-testdekking toegevoegd voor redaction, protocolcompatibiliteit, queue/playlist-normalisatie, crash flags, wakeword defaults, demo mode, permissions en mDNS lifecycle.
- Public unsigned release-ondersteuning verbeterd met Windows ARM64-artifacts, CI concurrency en fallback-afhandeling voor GitHub-hosted Mac Catalyst.
