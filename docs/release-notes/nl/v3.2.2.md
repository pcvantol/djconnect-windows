# DJConnect Windows 3.2.2

- Brengt Ask DJ en Track Insight in lijn met het Home Assistant DJConnect
  `3.2.2` contract.
- Gebruikt overal Music DNA-terminologie en verwijdert legacy memory-tekst
  en payloadvoorbeelden.
- Voegt directe Track Insight toe vanuit Speelt Nu en toont genormaliseerde
  `track_insight` resultaten, inclusief Music DNA Match, vibe hints en een
  empty state wanneer er geen track speelt.
- Stuurt Ask DJ Track Insight-intents naar dezelfde Track Insight-weergave
  zonder zelf playbackknoppen of oude media te verzinnen.
- Voegt een optionele lokale Home Assistant `/api/websocket` fast path toe voor
  ondersteunde commands, Ask DJ-berichten en Track Insight, met HTTP als
  canonieke fallback.
- Legt de Windows Track Insight-platformroadmap vast voor eigen navigatie,
  demo provider, Auto Track Insight en toekomstig visualizerwerk.
- Scherpt Windows app-pairing aan naar de outbound Home Assistant-flow: lokale
  Home Assistant URL, Home Assistant koppelcode en geen Windows clientadres,
  app-gegenereerde code of inbound wachtstatus.
- Vervangt het eerste welkomstscherm door een vijfstappen-tour voor Speelt Nu,
  Ask DJ, Track Insight, Wachtrij en Mini-games.
