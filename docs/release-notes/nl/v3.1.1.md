# DJConnect Windows 3.1.1

- Publicatie van niet-ondertekende Windows- en Mac Catalyst-releases toegevoegd naar de gedeelde DJConnect app releases-repository.
- Windows ARM64 diagnostic release-artifacts toegevoegd voor Windows on ARM, waaronder Parallels Windows-VM's op Apple Silicon Macs.
- Nederlandse en Engelse What's New-publicatie toegevoegd voor Windows en Mac Catalyst op djconnect.dev.
- MAUI build-inrichting voor Mac Catalyst gerepareerd, inclusief platform-entrypoint en documentatie voor side-by-side Xcode 26.4 builds.
- Windows MAUI platform-entrypoint en manifestbestanden toegevoegd voor CI/release builds.
- Release-permissiedocumentatie bijgewerkt voor GitHub Actions, releasepublicatie en workflow cleanup.
- Desktop-schermen toegevoegd voor Instellingen, Afspeellijsten, Privacy, Logs, Feedback en Crashrapporten.
- Privacy-safe diagnostics-redaction toegevoegd voor preview, kopiëren en GitHub issue-prefill.
- Session-only Demo Mode voorbeelddata toegevoegd en wakeword-UI-state achter een feature gate gezet tot er een echte listener is.
