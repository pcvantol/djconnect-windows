# DJConnect Windows 3.2.6

- Stuurt de juiste Windows appversie-metadata naar Home Assistant.
- Geeft de gekozen app-taal en locale mee met DJConnect-commando's, Ask DJ, voice en WebSocket-verzoeken.
- Brengt de client in lijn met Home Assistant `3.2.15` response-shapes.
- Verbetert Queue refresh zodat rijen echte teruggegeven tracks tonen met titel, artiest, album en artwork-metadata.
