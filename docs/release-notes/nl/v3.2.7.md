# DJConnect Windows 3.2.7

- Toont de Ask DJ generated-text spark alleen wanneer Home Assistant gegenereerde tekst expliciet markeert.
- Stuurt Music DNA-context, mood, taal en locale mee met Ask DJ- en Track Insight-verzoeken.
- Decodeert servergestuurde Music DNA mood- en energy-profielen zonder lokale profielgissingen.
- Houdt BPM en toonsoort uit Track Insight terwijl ondersteunde analyseblokken zichtbaar blijven.
