# DJConnect Windows 3.1.5

- De 3.1.x desktop feature release opnieuw uitgebracht waarbij Windows unsigned publish per runtime tijdens publish restore uitvoert.
- Mac Catalyst release-artifact gating aangescherpt naar Xcode 26.4.x, gelijk aan de requirement van de geïnstalleerde .NET 10 Mac Catalyst workload.
- Bevat het werk uit 3.1.2 voor onboarding, pairing, Speelt Nu, Ask DJ, Wachtrij, Afspeellijsten, Instellingen, Privacy, Logs, Feedback, Crashrapport, Demo Mode, Wakeword-gated UI en diagnostics-redaction.
