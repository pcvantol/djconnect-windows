# DJConnect Windows 3.1.0

- Eerste desktopclient-scaffold voor Windows en Mac Catalyst.
- Home Assistant pairing, status, Ask DJ history-sync en playback-commandostructuur toegevoegd.
- Veilige lokale tokenopslag toegevoegd via Windows Credential Manager of macOS Keychain.
- Server-side Ask DJ history ondersteund met revisiemetadata en clear-history flow.
- CI, protocoltests en niet-ondertekende diagnostic release-artifacts toegevoegd.
