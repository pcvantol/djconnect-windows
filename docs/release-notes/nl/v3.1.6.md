# DJConnect Windows 3.1.6

- De 3.1.x desktop feature release opnieuw uitgebracht met CI- en release-gating voor Mac Catalyst aangescherpt naar Xcode 26.4.x.
- Windows unsigned artifacts hersteld met één expliciete `win-x64;win-arm64` restore vóór no-restore publish.
- Bevat het werk uit 3.1.2 voor onboarding, pairing, Speelt Nu, Ask DJ, Wachtrij, Afspeellijsten, Instellingen, Privacy, Logs, Feedback, Crashrapport, Demo Mode, Wakeword-gated UI en diagnostics-redaction.
