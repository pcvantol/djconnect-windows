# DJConnect Windows 3.2.5

- Completes app-wide localization for English, Dutch, German, French and Spanish, including settings, alerts, onboarding, mini-games and empty states.
- Connects Windows `djconnect://` protocol activation to the existing QR/deeplink pairing flow.
- Adds stricter localization validation for missing keys, duplicate keys, placeholder mismatches and resource formatting.
- Polishes ordinary Windows UI copy so technical diagnostics stay in About, feedback and logs.
