# DJConnect Windows 3.1.2

- Added onboarding-gated local pairing with Client address, pairing code copy actions, stricter mDNS advertising and pairing reset.
- Added full desktop screens for Now Playing, Ask DJ, Queue, Playlists, Settings, Privacy, Logs, Feedback, Crash reports, About, Legal, What's New and Mini-games.
- Added privacy-safe diagnostics redaction for logs, feedback, crash reports, clipboard exports and GitHub issue prefill.
- Added session-only Demo Mode with local sample playback, queue, playlists and Ask DJ responses.
- Added Update Required handling for protocol mismatches while keeping settings, logs, privacy, legal and feedback available.
- Added wakeword prompt/settings state behind a feature gate until a real foreground listener is available.
- Added non-functional requirement coverage for redaction, protocol compatibility, queue/playlist normalization, crash flags, wakeword defaults, demo mode, permissions and mDNS lifecycle.
- Improved public unsigned release support with Windows ARM64 artifacts, CI concurrency and GitHub-hosted Mac Catalyst fallback handling.
