# DJConnect Windows 3.2.0

- Upgraded the Windows client to the DJConnect `3.2.x` protocol.
- Pairing now happens by entering the local Home Assistant URL and the code
  shown by the DJConnect integration. Windows no longer hosts a local Client API
  or advertises mDNS for pairing.
- After local pairing, the app can fall back from local Home Assistant to a
  supported remote Home Assistant URL.
- Added diagnostics for HA connection mode, remote support, music backend,
  backend revision, target player and backend capabilities.
- Playback actions are forwarded as backend-owned actions so Spotify Direct and
  Music Assistant responses can both work without Spotify-only assumptions.
