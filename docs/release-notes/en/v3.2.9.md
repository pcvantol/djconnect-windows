# DJConnect Windows 3.2.9

- Uses the canonical Home Assistant `/api/djconnect/v1/...` routes for pairing, status, Ask DJ, Track Insight, Music DNA and Music Discovery.
- Updates QR/deeplink pairing validation for the v1 pair endpoint while keeping the Windows identity contract unchanged.
- Adds a regression check so old non-v1 DJConnect API routes cannot slip back into client code or docs.
